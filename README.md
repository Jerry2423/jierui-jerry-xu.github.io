# Academic Website Template

This is a clean academic website template based on [Jon Barron's website](https://jonbarron.info/). The original content has been removed and replaced with placeholder templates for your own use.

## Getting Started

1. **Update personal information**: Edit `index.html` to replace placeholder text with your own information
2. **Add your profile photo**: Place your profile photo as `images/profile.jpg`
3. **Add your CV and bio**: Place your CV as `data/YourName-CV.pdf` and bio as `data/YourName-bio.txt`
4. **Add publication images**: Add images for your papers in the `images/` directory
5. **Update publications**: Replace the sample publication entries with your own research
6. **Customize styling**: Modify `stylesheet.css` if desired

## File Structure

- `index.html` - Main website file with placeholder content
- `stylesheet.css` - CSS styling (unchanged from original)
- `data/` - Directory for CV, bio, and bibliography files
- `images/` - Directory for profile photo and publication images
- `CNAME` - Domain configuration (update if using custom domain)

## Template Features

- Responsive design that works on desktop and mobile
- Hover effects for publication images
- Clean, professional academic layout
- Easy to customize and extend
